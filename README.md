# Macroeconometric Analysis of Oil Price Volatility

## Objective
Analyze oil price volatility using ARIMA-GARCH models with geopolitical risk.

## Variables
- Oil Prices (Dependent Variable)
- GPR Index
- VIX
- DXY
- Dummy Variables:
   - Global Financial Crisis (2008)
   - 2014 OIL Crash
   - COVID-19
   - Russia-Ukraine War

## Methodology
- Data Cleaning
- Unit Root Tests
- ARIMA
- GARCH Modeling
- Volatility Forecasting

## Tools Used
- Python
- Jupyter Notebook
- Pandas
- Statsmodels
- ARCH package
